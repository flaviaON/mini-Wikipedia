Python

Definição

Python é uma linguagem de programação de alto nivel. Imperativa, orientada a objetos,dinamica e etc...Atualmente possui um modelo de desenvolvimento comunitário Ela possui, apesar de algumas partes possuir um padrão, ela não é tão especifica Ela tem como objetivo legibilizar o codigo sobre velocidade e expressividade

Caracteristiscas

Uma dentre as varias caracteristicas de Python é a sua objetividade, assim democratiza o ensino. Ela é orientada a objetos e foi criada em 1991, a sua simplicidade é a sua maior caracteristica.

Ferramentas de Desenvolvimento

Para o desenvolvimento do projeto do projeto desenvolvido em Python é utilizado algumas ferramentas. Há ferramentas para editar o código-fonte (os IDE), que vão ajudar o programador a esconomizar tempo, pois as mesmas possuem ferramenta para preenchimento automatico, refatoração, compilador, debugger, geração de código, permitindo assim o aumento da produtividade na hora de desenvolverf alguma aplicação.
